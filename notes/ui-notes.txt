    - parent
  * Templates
    - P: props
    - S: CustomStyles

    - window (sdl2w Window)

    - children (UIElement unique ptr array)
    - stateInterface (optional StateInterface)
    - style
    - customStyle
  * Methods
    - setStateInterface
    - dispatchAction
    - dispatchUiAction
    - getEntityById
    - getTemplateById
  * Methods (Override)
    - setStyle

    - getCustomStyle
    - setProps
    - getProps

    - render

STYLES
Position
  - x
  - y
Size
  - width
  - height
Scale
  - scale
TextParams
  - font family (enum PARAGRAPH H1 H2 H3)
  - font size
  - font color (string)
  - align (enum LEFT CENTER RIGHT)
List
  - line height
  - line spacing
  - line background color

EVENTS
  - clickable (requires Position & Size)
  - hoverable (requires Position & Size)

ELEMENTS
Sprite
  * Description: Renders a stylized sprite.
  * Props
    - sprite (string)
  * Styles
    - Position, Size, Scale
Quad
  * Description: Renders a stylized rectangle with children on it.
  * Props
    - children (list if UiElements to be rendered on this quad)
  * Styles
    - Position, Size, Scale
    - bg color (string)
    - bg sprite (string)
    - border color (string)
    - border size
SectionDropShadow
  * Description: Renders a quad with an opinionated background.
  * Styles
    - Position, Size, Scale
    - bg color (string)
    - bg sprite (string)
    - drop shadow offset
    - drop shadow color
    - border color
    - border size
SectionScrollable

  * Styles
    - Position, Size, Scale
    - border color (string)
    - border size
  * Actions
    - UIScrollSectionTo
TextLine

  * Styles
    - Position, TextParams
TextParagraph

  * Styles
