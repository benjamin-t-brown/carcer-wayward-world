Terrain tileset generator

Loads 28×32 terrain tiles (grass, dirt, ocean, cave floor, snow). Pick primary (fill) and secondary (border) terrain; the tool composites border strips over primary using overlapSize (px). Right-click the output canvas → Copy image, then paste into your art editor. Tile order matches the 9+4 autotile strip used in terrain_borders.
